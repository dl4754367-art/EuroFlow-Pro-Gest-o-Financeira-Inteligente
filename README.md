### ✨ Sobre esta aplicação
O **EuroFlow Pro** é uma ferramenta leve e intuitiva desenhada para quem quer ter o controlo total do seu dinheiro sem complicações. Registre gastos, analise padrões e receba conselhos de uma IA integrada.

### 🚀 Funcionalidades Principais
* **🤖 Consultor Pro:** Dicas inteligentes baseadas no teu saldo.
* **🎯 Barra de Metas:** Define objetivos de poupança e vê o progresso visualmente.
* **📊 Exportação Excel:** Leva os teus dados para o Excel com fórmulas automáticas.
* **📱 Mobile First:** Funciona perfeitamente no telemóvel como uma App nativa.
